# 迭代 0.0.1 — 番茄钟计时器首版

## 范围

实现纯前端番茄钟计时器页面：25分钟倒计时、开始/暂停/重置按钮、归零视觉高亮。

## 源需求

- e000001: owner_input — 老板原始需求
- e000002: requirement_defined — Hermes 结构化需求
- e000007: item_added — pomodoro-ui 功能条目
- e000008: item_added — pomodoro-deploy 部署条目

## 技术方案

纯静态单文件 HTML+CSS+JS，无框架依赖。部署到 Aliyun Nginx 子路径 `/projects/pomodoro-reg/`。

## 功能清单

- [ ] 25:00 倒计时显示
- [ ] 开始按钮：启动倒计时
- [ ] 暂停按钮：暂停倒计时  
- [ ] 重置按钮：重置到 25:00
- [ ] 归零时视觉高亮变化

## 验收标准

- 加载页面显示 25:00
- 开始/暂停/重置交互正确
- 归零时页面明显视觉变化
- 公网可访问，Cache-Control: no-cache

## 排除

无自定义时长、无音效、无历史记录、无统计、无用户数据收集、无移动端适配

## 测试计划

- 浏览器加载验证
- 按钮交互功能测试
- 归零高亮视觉检查
- 公网验收
