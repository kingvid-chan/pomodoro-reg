## 技术方案
纯静态单页面应用，HTML + CSS + JavaScript，无框架、无依赖、无后端。

### 页面结构
- 单个 HTML 文件 `src/index.html`
- 内嵌 CSS 和 JS（单文件自包含）
- basePath 支持：页面须正确处理 `/projects/pomodoro-reg/` 子路径

### 核心功能
1. 25分钟倒计时显示（MM:SS 格式）
2. 开始按钮：启动倒计时
3. 暂停按钮：暂停倒计时
4. 重置按钮：重置到 25:00
5. 倒计时归零（00:00）时页面高亮变化（如背景色变绿、文字变大）

### 验收标准
- 倒计时数字从 25:00 逐秒递减
- 归零时页面有明显视觉变化（高亮）
- 开始/暂停/重置按钮功能正确
- 所有静态资源（CSS/JS/图片）URL 带版本令牌 `?v=0.0.1`，且路径前缀为 `/projects/pomodoro-reg/`
- 文档 HTTP 响应头含 `Cache-Control: no-cache`（需在部署时通过 Nginx 或 Flask 中间件实现）

### 数据边界
- 不使用 localStorage、cookie、sessionStorage
- 不收集任何用户数据
- 无登录/注册
- 无音效
- 无历史记录或统计
- 无移动端适配
- 不开放自定义时长
